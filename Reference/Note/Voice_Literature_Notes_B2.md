## 第8篇：王尧 & 李朋波 (2023)

### 基本信息
| 项目 | 内容 |
|------|------|
| **标题** | 不确定环境中员工建言对授权领导影响的机制研究 |
| **期刊** | 财经问题研究 |
| **样本** | 8家国企+78部门+297对领导-员工 |
| **理论** | 信任理论（认知信任+情感信任） |
| **评价** | 🟡 重要参考 |

### 核心贡献

**反向视角：员工建言→领导行为**

```
环境不确定性高
       ↓
员工建言 → 领导信任下属 → 授权领导↑
```

**关键发现**：
1. **环境不确定性是关键调节变量**
   - 高不确定性时：建言→信任↑→授权↑
   - 低不确定性时：关系不显著甚至可能负面

2. **建言可以赢得领导信任**
   - 认知信任：员工显示能力
   - 情感信任：员工显示忠诚

3. **视角转换**：从「领导如何影响员工建言」到「员工建言如何影响领导行为」

### 对我们研究的核心贡献

**与我们命题的联系**：

| 他们的变量 | 对应我们的变量/命题 |
|----------|------------------|
| 环境不确定性调节 | AI时代可视为高不确定环境 |
| 员工建言 | 向上影响行为 |
| 授权领导 | 上级行为改变（我们的隐含结果） |

**理论启示**：
1. Oc & Bashshur (2013)引用：「掌握他人需要的资源，是对他人产生影响的重要前提」—— **直接支持我们的资源依赖框架！**
2. AI时代 = 高不确定环境 → Z世代员工建言更可能获得信任和授权

**关键引用**：
> 「在高度不确定的环境中，员工相比于领导在资源（如资金、权力）上通常处于劣势地位…高度不确定的环境为员工建言对领导授权产生积极影响创造了有利的契机」

### 批判性评估

**优势**：
- 跨层模型设计（团队层+个体层）
- 三时点数据收集
- 明确的边界条件（环境不确定性）

**局限**：
- 样本全是国企
- 未考虑员工资源类型（如AI能力）
- 未考虑代际差异

## 第9篇：向常春 & 龙立荣 (2014) 🟡

### 基本信息
| 项目 | 内容 |
|------|------|
| **标题** | 参与型领导与员工建言：积极印象管理动机的中介作用 |
| **样本** | 231员工+77上级配对，广东生产型企业 |
| **理论** | 社会信息处理+人际期望理论 |
| **方法** | HLM跨层分析 |

### 核心贡献

**积极印象管理动机（AIM）只中介抑制型建言！**

```
参与型领导 ─┬─→ 心理安全感 ─→ 促进性/抑制性建言
            │
            └─→ 积极印象管理动机 ─→ 抑制性建言（仅此！）
```

**关键发现**：
1. **印象管理动机只影响抑制型建言**——促进性需要持续认知努力，AIM驱动的员工选择更"省力"的抑制型
2. 人际公平正向调节AIM与抑制型建言的关系

### 对我们研究的启示

**与我们「向上影响行为」的联系**：
- 本文支持建言作为印象管理工具的观点
- 我们可以探讨：Z世代员工是否更倾向于以AI资源作为印象管理手段？

**关键引用**：
> 「中国文化最突出的特征是强调社会的和谐与人际关系的融洽，作为保护面子、人情及关系的策略，印象管理不仅是必然的，也是被期望的」

## 第10篇：刘军等 (2023) 🟡

### 基本信息
| 项目 | 内容 |
|------|------|
| **标题** | 领导情绪表达怎样影响员工建言？ |
| **样本** | 310份两阶段数据，42家企业 |
| **理论** | EASI + SDT自我决定理论 |
| **方法** | 两阶段有调节的中介模型 |

### 核心贡献

**三种工作动机 × 三种建言 × 差序氛围调节**

**工作动机三维度**：
1. **绩效动机**：完成任务的动机
2. **面子动机**：维护组织自尊
3. **职业成长动机**：发展规划

**建言三维度**：
1. 促进型：改善组织
2. 抑制型：指出问题
3. **防御型**：维护个人利益（新维度！）

**关键发现**：
- 消极情绪表达→面子动机↑→防御型建言↑
- 差序氛围在「面子动机→建言」中起**负向调节**作用
- 积极情绪对防御型建言无显著影响

### 对我们研究的启示

**差序格局与我们研究的联系**：
- 高权力距离+差序格局 = 中国情境特殊性
- 当Z世代员工拥有AI资源优势时，可能改变差序氛围中的权力动态

**防御型建言的启示**：
- 即使是「向上影响行为」也可以是防御性的
- 我们的研究需要区分员工影响上级的不同动机

## 第11篇：周晔等 (2020) 🟡

### 基本信息
| 项目 | 内容 |
|------|------|
| **标题** | 资质过剩感会激发员工建言吗？——基于公平启发视角 |
| **样本** | 256银行柜员+47主管，华北商业银行 |
| **理论** | 公平启发理论 |
| **方法** | 三维交互模型（HLM） |

### 核心贡献

**资质过剩感 × 互动公平 × 工作年限 → 建言**

**关键发现**：
1. 资质过剩感和建言**无直接主效应**
2. **互动公平非线性调节**：
   - 高互动公平：资质过剩感 → 建言 ↑
   - 低互动公平：资质过剩感 → 建言 ↓
3. **工作年限削弱调节效应**：新员工更依赖互动公平信号

### 对我们研究的核心启示

**与我们「资源依赖」框架的联系**：
- 资质过剩 ≈ 员工拥有超出岗位的资源
- AI素养高的Z世代 = 某种意义上的「资质过剩」
- 互动公平 ≈ 上级开放性的一个维度

**关键引用**：
> 「资质过剩的员工由于个人才华不能得到充分展现...出于自我提升的目的，员工个体的负面情绪在特定的情境下可能激发建言这种积极的行为」

**边界条件启示**：
- 工作年限短 + 高互动公平 → 资质过剩感对建言的正向影响最强
- 这可能解释为什么Z世代（工作年限短）更可能利用AI资源进行向上影响

## 第12篇：Deng, Walter & Guan (2020) 🔴 **核心文献！**

### 基本信息
| 项目 | 内容 |
|------|------|
| **标题** | Supervisor-Directed Emotional Labor as Upward Influence: An EASI Perspective |
| **期刊** | Journal of Organizational Behavior |
| **样本** | 377下属+91上级，中国央企，三时点数据 |
| **理论** | EASI (Emotions as Social Information) |

### 核心贡献

**情绪劳动作为向上影响策略！**

```
Supervisor-Directed Emotional Labor
    │
    ├── Surface Acting (假装) ──→ 负面效果
    │       ↓
    │   - 降低Liking
    │   - 降低Competence感知
    │
    └── Deep Acting (真诚) ──→ 正面效果
            ↓
        - 提升Liking → 奖励推荐↑
```

**关键调节变量**：上级的**认知动机(Epistemic Motivation)**
- 高认知动机上级：更能识别情绪真伪，放大surface/deep acting的效果
- 低认知动机上级：对情绪劳动不敏感

### 对我们研究的核心贡献

**这是直接研究「向上影响」的文献！**

| 他们的概念 | 对应我们的概念 |
|----------|-------------|
| Supervisor-Directed Emotional Labor | 向上影响行为的一种 |
| Deep Acting (真诚情绪) | 基于真实资源的影响 |
| Surface Acting (假装情绪) | 可能类比为没有AI能力却假装懂 |
| Epistemic Motivation | 类似于上级开放性 (P4) |
| Reward Recommendations | 向上影响的结果 |

**理论启示**：
1. **向上影响的双刃剑**：真诚有效，假装有害
2. **上级特征调节**：并非所有上级都对下属影响策略敏感
3. **EASI模型**：情绪作为社会信息 → 可扩展到AI资源作为社会信息

**关键引用**：
> "Subordinates often strive to influence supervisors' impressions... to access organizational resources"

> "Supervisor-directed emotional labor [is] a novel means of upward influence"

### 批判性评估

**优势**：
- 三时点数据收集
- 上下级配对
- 双路径模型（情感+认知）
- 控制了传统印象管理策略

**局限**：
- 样本来自单一国企
- 未考虑员工持有的特定资源类型（如AI能力）

## 第13篇：蔡文著 & 余莉 (2024) 🟡

### 基本信息
| 项目 | 内容 |
|------|------|
| **标题** | 不得已者而后言：基于自我调节理论的员工建言新内涵 |
| **样本** | 实验(493) + 问卷(269)，银行+多行业 |
| **理论** | 自我调节理论 + 情绪认知评价理论 |
| **方法** | 实验+3时点问卷调查 |

### 核心贡献

**心理契约破裂 → 负面情绪 → 积极建言！**

**颠覆性发现**：
1. 心理契约破裂**正向**影响建言（而非负向）
2. 负面情绪（失望+懊悔）中介这一关系
3. 建言效能感正向调节负面情绪→建言路径

**关键理论创新**：
- 建言**不仅是**亲组织行为，**也可以是**员工维护自身利益的工具
- 负面情绪并不必然导致消极行为

### 对我们研究的启示

**与我们研究的潜在联系**：
- 当Z世代员工感知AI资源优势但未被组织认可时 ≈ 心理契约破裂
- 这种"不公平感知"可能激发向上影响行为
- 建言效能感 ≈ 我们可以纳入的边界条件

**关键引用**：
> 「员工在自身利益受到损害时所做出的建言是员工为了应对变化而采取的积极的适应性行为」

## 第14篇：王尧、罗文豪、侯亮 (2022) 🟡 研究构想

### 基本信息
| 项目 | 内容 |
|------|------|
| **标题** | 逆境中何以建言？成就目标视角下员工建言韧性的形成机制与作用效果 |
| **类型** | 研究构想（非实证） |
| **理论** | 成就目标理论 + 韧性过程视角 |

### 核心贡献

**建言韧性三维度框架**

```
建言韧性 = 过程视角
    │
    ├── (1) 积极评价建言逆境
    │       「失败是成功之母」
    │
    ├── (2) 主动调节情绪状态
    │       排解压力，保持活跃思维
    │
    └── (3) 努力调整建言行为
            反思原因，改进策略
```

**成就目标定向的作用**：
- 学习目标定向 → 建言韧性 ↑
- 成绩证明目标定向 → 建言韧性 ↑（但聚焦策略而非质量）
- 成绩回避目标定向 → 建言韧性 ↓

### 对我们研究的启示

**与Z世代的潜在联系**：
- Z世代被认为更具学习目标定向（追求自我提升）
- 高AI素养的Z世代可能更具建言韧性
- 这可能解释他们为何敢于向上影响

**方法论启示**：
- 韧性应从**过程视角**而非行为结果来定义
- 「向上影响」也可能是一种需要韧性的过程

| 我们的命题 | 支持文献 | 核心发现 |
|----------|---------|---------|

4. **定性访谈**：捕捉「向上影响」的情境特殊性
