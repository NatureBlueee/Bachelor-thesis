## 第一篇：钟柏昌、刘晓凡、杨明欢 (2024)

### 基本信息

| 项目 | 内容 |
|------|------|
| **标题** | 何谓人工智能素养：本质、构成与评价体系 |
| **期刊** | 华东师范大学学报（教育科学版） |
| **DOI** | 10.16382/j.cnki.1000-5560.2024.01.005 |
| **评价** | 🔴 直接可用于论文 |

### 核心贡献

**1. 本质界定：人的技术化**

钟等从技术本体论视角分析AI素养的本质——**人的技术化**。这不是简单的"学会用AI工具"，而是"人在与技术的双向建构中获得本质力量的提升"。

引用马克思实践哲学：
- **主体客体化**：人创造技术（本质力量对象化于技术）
- **客体主体化**：技术反过来内化为人的本质力量

关键论断：
> "人的技术化仍然是人类通达自由与解放的合理途径...作为适应和驾驭人工智能的素养，它的本质就是通过人的技术化过程，获得人的本质力量的提升，最终实现人与智能技术的和谐发展。"

**2. 构成框架：知识-情感-思维三维**

| 维度 | 定义 | 二级指标 |
|------|------|----------|
| **AI知识** | 认知基础 | 理解与体验、系统设计与开发、系统评价与优化 |
| **AI情感** | 情感涵养 | AI与人类、AI与社会、交流与合作 |
| **AI思维** | 高阶能力 | 工程设计思维、计算思维 |

理论基础：
- 知识与思维的内在一致性（皮亚杰发生认识论）
- 认知与情感的双螺旋结构（布卢姆教育目标分类）

**3. 评价体系**

详细的三级指标体系（见原文表2-4），可作为我们研究中AI素养测量的理论参照。

### 对我们研究的贡献

**直接支撑P1（AI素养→感知资源优势）：**
- 提供了AI素养的规范性定义和三维框架
- 我们在Introduction和Literature Review中直接采用此框架
- 注意：原框架面向教育情境，我们需论证其在职场情境的适用性

**潜在启发：**
- "人的技术化"概念可丰富对Z世代与AI关系的理论理解
- 知识-情感-思维框架可指导访谈提纲设计（分维度探索）

### 批判性评估

**优势：**
- 理论根基深厚（皮亚杰、马克思、布卢姆）
- 框架完整、指标清晰
- 中文原创，直接可用

**局限：**
- 面向K-12教育，非职场情境
- 评价体系复杂（48项指标），不适合直接作为量表
- 缺乏实证检验

## 第二篇：付春香、侯庭轩 (2025)

### 基本信息

| 项目 | 内容 |
|------|------|
| **标题** | 人工智能冲击意识对Z世代员工主动性创新行为的影响：一个有调节的链式中介模型 |
| **期刊** | 经济管理 |
| **DOI** | 10.12181/jjgl.2025.02.05 |
| **评价** | 🔴 直接可用于论文 |

### 核心贡献

**1. 研究问题高度相关**

- 自变量：人工智能冲击意识（AI shock awareness）
- 因变量：主动性创新行为
- 样本：Z世代员工（N=380）
- 情境：中国制造业、快递业、房地产业

**2. 理论框架**

基于资源保存理论（COR）和压力认知评价理论：
- AI冲击意识 → 工作不安全感 → 工作旺盛感 → 主动性创新行为
- 容错动态性调节工作旺盛感→创新行为关系

**3. 关键发现**

| 假设 | 路径 | 结果 |
|------|------|------|
| H1 | AI冲击意识 → 主动性创新行为 | β=0.158** ✓ |
| H2 | AI冲击意识 → 工作不安全感 | β=0.223*** ✓ |
| H3 | 工作不安全感中介 | 效应值=0.029 ✓ |
| H4 | AI冲击意识 → 工作旺盛感 | β=0.179*** ✓ |
| H5 | 工作旺盛感中介 | 效应值=0.031 ✓ |
| H6 | 链式中介 | 效应值=0.011 ✓ |
| H7 | 容错动态性调节 | β=0.122* ✓ |

**4. 理论逻辑**

> "当Z世代员工将压力源视为挑战而非威胁时，他们的学习动机和工作活力将被激发...促使其以更积极的态度应对人工智能技术带来的威胁。"

### 对我们研究的贡献

**支撑整体框架：**
- 证明AI相关感知可正向影响Z世代员工行为（不只是焦虑/离职意愿）
- 资源保存理论与我们的资源依赖理论可形成对话
- "主动性创新行为"与我们的"向上影响行为"有概念重叠

**支撑P4（上级开放性调节）：**
- "容错动态性"作为组织氛围变量调节行为转化
- 类似于我们的"上级开放性"作为边界条件
- 可引用其调节效应的实证证据

**支撑方法论：**
- Z世代样本获取策略（筛选29岁以下）
- 行业选择依据（智能化转型快的行业）

### 批判性评估

**优势：**
- 样本高度相关（Z世代、中国情境）
- 实证数据扎实（N=380，路径系数显著）
- 2025年新发表，时效性强

**局限：**
- 自变量是"AI冲击意识"（威胁感知），不是"AI素养"（能力）
- 因变量是"主动性创新行为"，不是"向上影响行为"
- 横截面数据，无法确定因果

**与我们研究的区别：**
| 维度 | 付春香等(2025) | 本研究 |
|------|----------------|--------|
| 自变量 | AI冲击意识（威胁感知） | AI素养（能力） |
| 因变量 | 主动性创新行为 | 向上影响行为 |
| 中介 | 工作不安全感、工作旺盛感 | 感知资源优势 |
| 调节 | 容错动态性 | 上级开放性 |
| 方法 | 定量问卷 | 定性访谈 |

## 第三篇：后Morrison (2011) 以来员工建言研究的演进与整合

### 基本信息

| 项目 | 内容 |
|------|------|
| **标题** | 后Morrison (2011) 以来员工建言研究的演进与整合 |
| **性质** | AI辅助生成的二级综述（基于Consensus对话） |
| **评价** | 🔴 直接可用于论文（整合性高，Z世代视角独特） |

### 核心贡献

**1. Morrison模型的修正与拓展**

原模型（2011）：安全性 + 效能感 → 建言决策

后续修正：
- **Liang等(2012)**：区分促进性建言 vs 禁止性建言
- **Maynes & Podsakoff(2014)**：加入破坏性建言、防御性建言
- **Aryee等(2017)**：双路径机制——接近动机 > 回避动机
- **Hsiung & Tsai(2017)**：高权力距离下安全感作用失灵
- **Hussain等(2019)**：旁观者效应——信息冗余抑制建言

**2. Z世代建言的独特机制**

文献提出了一个Z世代版的心理决策公式：

```
建言倾向_Z = a×安全感 + b×效能预期 - c×容忍延迟
```

其中：
- `a` 相对较小（Z世代不那么受恐惧支配）
- `b` 相对较大（Z世代更重视"说了有没有用"）
- `c` 值高（Z世代"不改就走"的低耐心）

**3. 数字化渠道的风险重构**

| 渠道特征 | 对风险感知的影响 |
|----------|------------------|
| 匿名性 | ↓ 降低直接风险 |
| 群体支持 | ↓ 降低被针对风险 |
| 广泛传播 | ↑ 增加声誉/法律风险 |
| 即时反馈 | ↔ 快速验证效能 vs 公开冲突 |

**4. 未竟之业（博士选题战略机会）**

| 已饱和领域 | 战略机会区 |
|------------|------------|
| 建言类型划分 | 员工沉默深层原因 |
| 领导反应与组织后果 | 文化×代际交互 |
| 建言渠道/策略 | 新兴工作形态（远程、AI、零工） |

### 对我们研究的贡献

**直接支撑P2-P3（感知资源优势→向上影响）：**
- 建言行为本质上是向上影响行为的一种形式
- 我们的"向上影响"比传统"建言"更广：包括想法生成、推广、实施
- 可借鉴促进性/禁止性建言的二分来分析AI素养驱动的不同类型影响

**直接支撑P4（上级开放性调节）：**
- 大量证据表明领导开放性是建言→效果的关键调节
- 高权力距离情境下，安全感作用可能失灵→上级开放性更关键
- Zhang等(2023)直接检验了代际差异：Z世代更依赖环境安全信号

**支撑中国情境论述：**
- 高权力距离文化下的建言机制不同
- Z世代"整顿职场"现象的学术解读
- 数字化渠道（社交媒体吐槽）作为替代性发声途径

### 批判性评估

**优势：**
- 整合性强，覆盖2011-2025核心文献
- 直接讨论Z世代，与我们研究高度相关
- 提出可检验的理论推测（效能>恐惧）

**局限：**
- AI辅助生成的二级综述，非peer-reviewed原创研究
- 部分引用需回溯原文验证
- Z世代建言公式是推测性的，缺乏实证检验

**关键引文追踪清单：**
- [ ] Liang, Farh & Farh (2012, JAP) — 促进性/禁止性建言
- [ ] Zhang et al. (2023, Scientific Reports) — 代际差异调节
- [ ] Morrison (2023, Annual Review) — 十年进展回顾

## 第四篇：周建涛、廖建桥 (2012)

### 基本信息

| 项目 | 内容 |
|------|------|
| **标题** | 为何中国员工偏好沉默——威权领导对员工建言的消极影响 |
| **期刊** | 管理世界 |
| **评价** | 🟡 间接参考价值 |

### 核心贡献

**1. 研究模型**

威权领导 → 心理授权(-)→ 员工建言(-)

调节变量：员工权力距离

**2. 关键发现**

- 威权领导负向影响心理授权（β=-0.38, p<0.001）
- 心理授权正向影响建言（β=0.21, p<0.01）
- 中介效应显著：威权领导通过削弱心理授权抑制建言
- 权力距离调节：高权力距离员工受威权领导影响更小

**3. 理论贡献**

- 解释了中国情境下"沉默偏好"的领导因素
- 心理授权作为中介机制——员工需要感到有能力、有意义才会建言
- 权力距离调节效应有趣：高PD员工反而更适应威权环境

### 对我们研究的贡献

**支撑P4（上级开放性调节）：**
- 威权领导 vs 开放型领导形成对照
- 证明领导风格是建言行为的关键边界条件
- 心理授权概念与我们的"感知资源优势"有关联

**支撑中国情境论述：**
- 直接解释为何中国员工倾向沉默
- 权力距离文化影响机制

### 批判性评估

**局限：**
- 2012年发表，较早
- 研究的是传统员工，非Z世代
- Z世代可能对威权领导反应不同（更可能离职而非沉默）

## 第五篇：李方君、祝养浩、赖斌 (2020)

### 基本信息

| 项目 | 内容 |
|------|------|
| **标题** | 员工建言发生和领导者建言采纳：权力感的视角 |
| **期刊** | 心理科学进展 |
| **评价** | 🔴 直接可用于论文 |

### 核心贡献

**1. 双阶段理论模型**

```
阶段1: 员工权力感 → 建言生成 → 建言表达
阶段2: 领导权力感 → 建言倾听 → 建言采纳
```

**2. 核心观点**

- **员工权力感**激活建言的"方法路径"→ 产生想法并敢于表达
- **领导权力感**可能激活"防御路径"→ 拒绝采纳（高权力者低估建议质量）

**3. 权力感的调节机制**

| 角色 | 权力感高 | 权力感低 |
|------|----------|----------|
| 员工 | 更敢建言、更主动 | 沉默、退缩 |
| 领导 | 可能拒绝采纳 | 更开放听取 |

### 对我们研究的贡献

**直接支撑整体框架：**
- "权力感"概念与我们的"感知资源优势"高度相关
- AI素养可能通过提升员工权力感促进向上影响
- 资源→权力感→影响行为的逻辑链

**支撑P4（上级开放性调节）：**
- 领导高权力感可能抑制建言采纳
- 上级开放性 ≈ 领导低防御性（抵消高权力感的负面效应）

## 第六篇：赵琛徽、汤昊 (2025)

### 基本信息

| 项目 | 内容 |
|------|------|
| **标题** | 人工智能技术应用对员工建言和沉默的影响：工作反刍和工作自主性的作用 |
| **期刊** | 科技进步与对策 |
| **DOI** | 10.6049/kjjbycd.D22024120437 |
| **评价** | 🔴 **极其相关！直接涉及AI+建言** |

### 核心贡献

**1. 核心命题：AI技术应用的"双刃剑"效应**

基于压力认知激活理论(CATS)：
- 路径1: AI应用 → 问题解决沉思(+) → 建言(+) / 沉默(-)
- 路径2: AI应用 → 情感反刍(+) → 建言(-) / 沉默(+)

**2. 工作自主性的调节作用**

| 工作自主性 | 对问题解决沉思 | 对情感反刍 |
|------------|----------------|------------|
| 高 | 强化AI→沉思→建言 | 弱化AI→反刍→沉默 |
| 低 | 弱化正向路径 | 强化负向路径 |

**3. 实证方法（双重验证）**

- 研究一：情景实验（N=212）
- 研究二：三阶段问卷（N=424）
- 所有假设得到支持

**4. 关键数据**

| 路径 | 效应值 | 
|------|--------|
| AI应用→问题解决沉思 | β=0.408*** |
| 问题解决沉思→建言 | β=0.345*** |
| 问题解决沉思→沉默 | β=-0.430*** |
| AI应用→情感反刍 | β=0.337*** |
| 工作自主性调节问题解决沉思 | β=0.324*** |
| 工作自主性调节情感反刍 | β=-0.258*** |

### 对我们研究的贡献

**直接支撑P1-P2（AI素养→感知资源优势→向上影响）：**
- 证明AI相关因素可以促进员工建言
- "问题解决沉思"是积极认知机制——类似于我们的"感知资源优势"
- AI应用产生的积极结果预期→适应性心理反应

**支撑P4（上级开放性调节）：**
- "工作自主性"作为边界条件类似于"上级开放性"
- 资源丰富（高自主性）时，AI应用的正向效应更强

**重要区别：**
| 维度 | 赵琛徽等(2025) | 本研究 |
|------|----------------|--------|
| 自变量 | AI技术应用（行为/环境） | AI素养（个人能力） |
| 因变量 | 建言/沉默 | 向上影响行为 |
| 中介 | 工作反刍（认知） | 感知资源优势（感知） |
| 调节 | 工作自主性 | 上级开放性 |
| 理论 | 压力认知激活理论 | 资源依赖理论 |

### 批判性评估

**优势：**
- 2025年最新发表，直接涉及AI+建言
- 双刃剑视角全面
- 情景实验+问卷双重验证

**局限：**
- AI技术应用≠AI素养（环境vs个人）
- 未区分Z世代
- 工作反刍机制与我们的资源优势机制不同
