# Reference 文件夹

## 设计目的

存放学术文献全文，是论文论据的根基和知识库。

## 内容结构

- `_INDEX.md` - 文献注册表
- 文献文件（PDF/MD 等格式）

## 使用规则

1. **添加文献**：将文献文件放入此文件夹，并在 `_INDEX.md` 中注册
2. **引用格式**：所有引用必须符合 **APA 第7版** 规范
3. **阅读要求**：每次阅读需通透理解，文献是唯一真相来源
4. **文献来源**：
   - Consensus 回答中提到的重要文献
   - 用户主动补充的文献
   - 论文写作过程中发现需要的文献

## 示例

查找某文献：在 `_INDEX.md` 中按作者/关键词搜索

引用文献：使用 `_INDEX.md` 中记录的 APA 格式引用
