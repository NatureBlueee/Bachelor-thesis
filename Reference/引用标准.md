# GB/T 7714-2015 参考文献著录规则完整说明

## 一、标准基本信息

**标准名称**：信息与文献 参考文献著录规则  
**标准号**：GB/T 7714-2015  
**实施日期**：2015年12月1日  
**替代标准**：GB/T 7714-2005  
**适用对象**：著者和编辑著录参考文献（非图书馆员或索引编制者使用）

---

## 二、核心概念

### 1. 参考文献定义
对信息资源或其中一部分进行准确和详细著录的数据，位于文末或文中。

### 2. 两种参考文献类型
- **阅读型参考文献**：著者为撰写论著而阅读过的信息资源，或供读者进一步阅读的信息资源
- **引文参考文献**：著者为撰写论著而引用的信息资源

---

## 三、两种标注体系

### **体系1：顺序编码制**（最常用）

**原理**：按引用文献在正文中出现的先后顺序编号，用方括号标注

**正文标注方式**：
```
示例1：德国学者研究了断裂[235]；之后又描述了构造带[236]。

示例2：多篇文献用逗号：裴伟[570,83]提出……

示例3：连续序号用短横线：莫拉德的研究[255-256]……

示例4：多次引用同一文献，标注首次序号+页码：
改变社会规范可能存在"二阶囚徒困境"问题[1]。
……苏格拉底、柏拉图和亚里士多德都是民主制度的反对者[2]260。
……佛教受到极大打击[2]326-329。
```

**参考文献表**：按序号顺序排列
```
[1] 著者. 题名[文献类型]. 出版地: 出版者, 年份: 页码.
[2] 著者. 题名[文献类型]. 出版地: 出版者, 年份.
[3] ...
```

---

### **体系2：著者-出版年制**

**原理**：标注著者姓氏和出版年，参考文献表按著者字顺和年份排列

**正文标注方式**：
```
示例1：已有研究探讨了这个问题(Crane, 1972)。

示例2：王临惠(2010a)指出……

示例3：多次引用加页码：
主编思想指挥全局已是共识(张忠智, 1997)，然而对编辑思想至今没有明确界定……
"思想"的内涵是"客观存在反映在人的意识中经过思维活动而产生的结果"(中国社会科学院语言研究所词典编辑室, 1996)1194。
```

**参考文献表**：按著者字母顺序+年份排列
```
陈登原, 2000. 国史旧闻: 第1卷[M]. 北京: 中华书局: 29.
张忠智, 1997. 科技书刊的总编...[C]//...论文汇编. 北京: ...: 33-34.
CRANE D, 1972. Invisible college[M]. Chicago: Univ. of Chicago Press.
```

## 五、著录格式详解

### （一）专著（图书）

**完整格式**：
```
主要责任者. 题名: 其他题名信息[文献类型标识]. 其他责任者. 版本项. 出版地: 出版者, 出版年: 引文页码[引用日期]. 获取路径. DOI.
```

**实例**：
```
[1] 陈登原. 国史旧闻: 第1卷[M]. 北京: 中华书局, 2000: 29.

[2] 哈里森, 沃尔德伦. 经济数学与金融数学[M]. 谢远涛, 译. 北京: 中国人民大学出版社, 2012: 235-236.

[3] 库恩. 科学革命的结构: 第4版[M]. 金吾伦, 胡新和, 译. 2版. 北京: 北京大学出版社, 2012.

[4] 赵学功. 当代美国外交[M/OL]. 北京: 社会科学文献出版社, 2001[2014-06-11]. http://www.cadal.zju.edu.cn/book/trySinglePage/33023884/1.
```

---

### （二）期刊文章

**完整格式**：
```
析出文献主要责任者. 析出文献题名[J]. 期刊名称, 年, 卷(期): 页码[引用日期]. 获取路径. DOI.
```

**实例**：
```
[1] 袁训来, 陈哲, 肖书海, 等. 蓝田生物群: 一个认识多细胞生物起源和早期演化的新窗口[J]. 科学通报, 2012, 55(34): 3219.

[2] 李炳穆. 韩国图书馆法[J/OL]. 图书情报工作, 2008, 52(6): 6-12[2013-10-25]. http://www.docin.com/p-400265742.html.

[3] DESMARAIS D J, STRAUSS H, SUMMONS R E, et al. Carbon isotope evidence for the stepwise oxidation of the Proterozoic environment[J]. Nature, 1992, 359: 605-609.
```

**年卷期标注方式**：
- `2012, 55(34): 3219` — 年，卷，期，页码
- `2014, 510: 356-363` — 年，卷，页码（无分期）
- `2010(6): 23` — 年，期，页码（无分卷）

---

### （三）报纸文章

**完整格式**：
```
析出文献主要责任者. 析出文献题名[N]. 报纸名称, 出版日期(版次).
```

**实例**：
```
[1] 丁文详. 数字革命与竞争国际化[N]. 中国青年报, 2000-11-20(15).

[2] 刘裕国, 杨柳, 张洋, 等. 雾霾来袭, 如何突围[N/OL]. 人民日报, 2013-01-12[2013-11-06]. http://paper.people.com.cn/rmrb/html/2013-01/12/nw.D110000renmrb_20130112_2-04.htm.
```

---

### （四）专著中的析出文献

**完整格式**：
```
析出文献主要责任者. 析出文献题名[文献类型]//专著主要责任者. 专著题名. 版本. 出版地: 出版者, 出版年: 析出文献页码.
```

**实例**：
```
[1] 程根伟. 1998年长江洪水的成因与减灾对策[M]//许厚泽, 赵其国. 长江流域洪涝灾害与科技对策. 北京: 科学出版社, 1999: 32-36.

[2] 马克思. 政治经济学批判[M]//马克思, 恩格斯. 马克思恩格斯全集: 第35卷. 北京: 人民出版社, 2013: 302.
```

---

### （五）学位论文

**完整格式**：
```
责任者. 题名[D]. 保存地: 保存单位, 年份: 页码.
```

**实例**：
```
[1] 马欢. 人类活动影响下海河流域典型区水循环变化分析[D/OL]. 北京: 清华大学, 2011: 27[2013-10-14]. http://www.cnki.net/...

[2] 吴云芳. 面向中文信息处理的现代汉语并列结构研究[D]. 北京: 北京大学, 2003.
```

---

### （六）专利文献

**完整格式**：
```
专利申请者或所有者. 专利题名: 专利号[P]. 公告日期或公开日期.
```

**实例**：
```
[1] 张凯军. 轨道火车及高速轨道火车紧急安全制动辅助装置: 201220158825.2[P]. 2012-04-05.

[2] 西安电子科技大学. 光折变自适应光外差探测方法: 01128777.2[P/OL]. 2002-03-06[2002-05-28]. http://211.152.9.47/sipoasp/zlijs/...
```

---

### （七）标准文献

**完整格式**：
```
主要责任者. 标准名称: 标准号[S]. 出版地: 出版者, 出版年: 页码.
```

**实例**：
```
[1] 全国信息与文献标准化技术委员会. 信息与文献 都柏林核心元数据元素集: GB/T 25100-2010[S]. 北京: 中国标准出版社, 2010: 2-3.

[2] 国家环境保护局科技标准司. 土壤环境质量标准: GB 15616-1995[S/OL]. 北京: 中国标准出版社, 1996: 2-3[2013-10-14]. http://wenku.baidu.com/...
```

---

### （八）电子资源

**完整格式**：
```
主要责任者. 题名[EB/OL]. (更新日期)[引用日期]. 获取路径.
```

**实例**：
```
[1] 萧钰. 出版业信息化迈入快车道[EB/OL]. (2001-12-19)[2002-04-15]. http://www.creader.com/news/20011219/200112190019.html.

[2] Dublin core metadata element set: version 1.1[EB/OL]. (2012-06-14)[2014-06-11]. http://dublincore.org/documents/dces/.
```

## 六、著录细则

### 1. 责任者著录规则

**个人著者**：
- 中文：姓名全称，如"李时珍"
- 汉语拼音：姓全大写，名可缩写，如"LI J N"或"LI Jiangning"
- 欧美著者：姓在前名在后，名可缩写，如"EINSTEIN A"
- 中译名：只著录姓，如"昂温"；同姓不同名时加名首字母，如"昂温 G"

**著者数量**：
- ≤3个：全部著录
- \>3个：著录前3个+", 等"（中文）或", et al."（外文）

**机关团体**：
- 中文：直接著录，如"中国科学院物理研究所"
- 外文：分级著录用"."连接，如"Stanford University. Department of Civil Engineering"

---

### 2. 题名著录规则

- 按信息源所载内容著录
- 合订题名：著录前3个，用";"分隔
- 副题名：用": "引出，如"化学动力学和反应器原理"
- 分卷书名：用": "引出，如"三松堂全集: 第4卷"
- 不著录并列题名（如中英对照题名只著录一个）

---

### 3. 版本著录

- 第1版不著录
- 其他版本：中文用"3版"，外文用"5th ed."或"Rev. ed."
- 古籍：可著录"刻本""抄本""明刻本"等

---

### 4. 出版项著录

**出版地**：
- 著录城市名称
- 同名异地城市加限定，如"Cambridge, Mass."和"Cambridge, Eng."
- 多个出版地只著录第一个
- 无出版地：中文"[出版地不详]"，外文"[S.l.]"

**出版者**：
- 可按原形式或缩写形式著录
- 多个出版者只著录第一个
- 无出版者：中文"[出版者不详]"，外文"[s.n.]"

**出版年**：
- 用公元纪年，阿拉伯数字
- 其他纪年加括号，如"1947(民国三十六年)"
- 报纸：YYYY-MM-DD格式，如"2013-01-08"
- 无法确定：用版权年"c1988"、印刷年"1995印刷"或估计年"[1936]"

---

### 5. 页码著录

- 专著或期刊中析出文献：著录页码
- 阅读型参考文献：著录起讫页或起始页
- 引文参考文献：著录引用信息所在页
- 引自序言或扉页：按实际情况著录，如"序2-3"或"扉页"

---

### 6. 电子资源特殊项

**引用日期**：
- 格式：[YYYY-MM-DD]
- 示例：[2013-03-20]

**获取路径**：
- 著录完整URL
- 示例：http://www.example.com/...

**DOI**（数字对象唯一标识符）：
- 如URL中已含DOI可省略单独著录
- 否则著录：DOI: 10.xxxx/xxxxx

---

### 7. 著录用符号

| 符号 | 用法 |
|-----|------|
| . | 分隔主要著录项目 |
| : | 引出其他题名信息、出版者、页码、专利号 |
| , | 分隔同一著作方式的责任者、"等""译"、出版年 |
| ; | 分隔合订题名、期刊后续年卷期 |
| // | 引出析出文献的出处 |
| ( ) | 期刊期号、报纸版次、非公元纪年 |
| [ ] | 文献序号、文献类型标识、引用日期、自拟信息 |
| / | 文献载体标识前、合期期号间 |
| - | 起讫序号和起讫页码间 |

## 七、实际应用示例

### 示例1：引用单篇期刊文章（顺序编码制）

**正文**：
```
袁训来等人发现蓝田生物群为研究多细胞生物起源提供了新窗口[1]。
```

**参考文献表**：
```
[1] 袁训来, 陈哲, 肖书海, 等. 蓝田生物群: 一个认识多细胞生物起源和早期演化的新窗口[J]. 科学通报, 2012, 55(34): 3219.
```

---

### 示例2：引用图书（顺序编码制）

**正文**：
```
陈登原在《国史旧闻》中记载……[1]29。
```

**参考文献表**：
```
[1] 陈登原. 国史旧闻: 第1卷[M]. 北京: 中华书局, 2000: 29.
```

---

### 示例3：引用多篇文献（顺序编码制）

**正文**：
```
多项研究表明[1-3,5]……
```

---

### 示例4：著者-出版年制

**正文**：
```
已有研究(Crane, 1972; Baker and Jackson, 1995)表明……王临惠(2010a, 2010b)的研究发现……
```

**参考文献表**：
```
BAKER S K, JACKSON M E, 1995. The future of resource sharing[M]. New York: The Haworth Press.
CRANE D, 1972. Invisible college[M]. Chicago: Univ. of Chicago Press.
王临惠, 2010a. 天津方言的源流关系刍议[J]. 山西师范大学学报(社会科学版), 37(4): 147.
王临惠, 2010b. 从几组声母的演变看天津方言形成的自然条件和历史条件[C]//曹志耘. 汉语方言的地理语言学研究. 北京: 北京语言大学出版社: 138.
```

## 八、常见问题处理

### 1. 无责任者
- 顺序编码制：可省略此项，直接著录题名
- 著者-出版年制：标注"佚名"或"Anon"

### 2. 责任者超过3个
著录前3个+", 等"或", et al."

### 3. 同一著者同年多篇文献
出版年后加小写字母区分：2010a, 2010b, 2010c...

### 4. 多次引用同一文献
- 顺序编码制：标注首次序号+"[]"外角标页码
- 著者-出版年制：标注著者年份+"()"外角标页码

### 5. 电子资源必备项
- 文献类型/载体标识：如[M/OL]
- 引用日期：[YYYY-MM-DD]
- 获取路径：http://...

---

## 九、快速检查清单

在完成参考文献著录后，请检查：

✓ 是否选择了正确的标注体系（顺序编码制或著者-出版年制）  
✓ 文献类型标识是否正确（M, J, D, P等）  
✓ 电子资源是否加了载体标识（/OL, /CD等）  
✓ 责任者姓名格式是否规范  
✓ 符号使用是否正确（., : , ; // 等）  
✓ 出版年格式是否统一  
✓ 页码是否著录（析出文献必须有）  
✓ 电子资源是否有引用日期和URL  
✓ 多作者是否用了"等"或"et al."  
✓ 参考文献表顺序是否正确

---

**注意事项**：
1. 本标准适用于各学科论文、著作的参考文献著录
2. 学位论文、期刊投稿可能有特殊要求，需以目标单位要求为准
3. 著录时应以文献本身信息源为准，保持准确性
4. 网络资源务必注明引用日期，因为内容可能变化
