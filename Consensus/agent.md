# Consensus 文件夹

## 设计目的

记录与 Consensus App（学术AI）的问答，Consensus 连接 SCI 学术文献库，是获取文献支持和学术洞见的重要渠道。

## 内容结构

- `_INDEX.md` - 问答注册表
- `_TEMPLATE.md` - 问答模板
- `CON-XXXX.md` - 具体问答记录

## 使用规则

1. **提问原则**：
   - 问题必须**具体而深入**，有学术牙齿
   - **不预设答案**，目的是探索和学习
   - 准备接受批判，唯一目标是发现事实
2. **索引编码**：`CON-0001`, `CON-0002`...（一个文件可含多个相关问题）
3. **注册要求**：新问答必须在 `_INDEX.md` 中注册
4. **文献追踪**：回答中提到的新文献，若 Reference 未收录，需标记并提出收集需求

## 示例

**好的问题**：
"在组织行为学领域，AI 素养（AI literacy）与员工向上影响策略选择之间的关系机制有哪些实证研究？这些研究使用了什么样的测量工具？"

**不好的问题**：
"AI 素养对员工有什么影响？"（太宽泛，无法获得有牙齿的回答）
