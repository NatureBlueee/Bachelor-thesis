# 开题报告答辩 PPT 设计方案

## AI素养对Z世代员工向上影响行为的影响：资源依赖视角

---

## 设计理念

### 视觉主题
- **配色方案**：
  - 主色：深蓝 (#1E3A8A) - 代表学术严谨
  - 辅色：渐变紫蓝 (#6366F1 → #3B82F6) - 代表AI和创新
  - 强调色：橙色 (#F59E0B) - 用于关键数据和结论
  - 背景：浅灰白 (#F9FAFB)

- **字体建议**：
  - 标题：思源黑体 Heavy / 微软雅黑 Bold
  - 正文：思源黑体 Regular / 微软雅黑 Regular
  - 数据/关键词：思源黑体 Medium（加粗）

- **视觉风格**：
  - 极简主义（每页不超过3个核心元素）
  - 图标化表达（用icon代替长文字）
  - 渐进式动画（要点逐条出现，不要一次性全显示）

### 【第1页】封面 - Title Slide

**视觉设计**：
```
┌─────────────────────────────────────┐
│                                     │
│        [AI芯片图标或神经网络图]        │
│                                     │
│   AI素养对Z世代员工向上影响行为的影响   │
│      —— 资源依赖视角                 │
│                                     │
│   The Impact of AI Literacy on      │
│   Upward Influence Behavior among   │

│                                     │
│   汇报人：XXX                        │
│   指导教师：XXX                      │

│   2024年12月                         │
│                                     │
└─────────────────────────────────────┘
```

**设计要点**：
- 背景用渐变（深蓝→浅蓝），营造科技感

- 标题分两行，中文大，英文小
- 底部信息居中对齐

**演讲提示**：
- 不用念标题，直接说："各位老师好，我今天汇报的核心问题是——当年轻员工比老板更会用AI的时候，会发生什么？"

- "数据分析用主题分析法，遵循经典的六步流程。"

### 【第13页】为什么用定性方法？

**视觉设计**：
```
┌─────────────────────────────────────┐
│   为什么用定性研究？                 │
│                                     │
│   ❓ vs 💯                          │
│   定性研究   vs   量化研究           │
│                                     │
│   我们的选择：定性                   │
│                                     │
│   ✅ 现象太新（ChatGPT才2年）        │
│      → 没有成熟量表                 │
│                                     │
│   ✅ 我们想知道"怎么发生"            │
│      → 不只是"有没有关系"           │
│                                     │
│   ✅ 需要捕捉情境化的过程            │
│      → 量表可能错过细节              │
│                                     │
│   ✅ 为未来量化研究打基础            │
│      → 探索→验证 的研究路径         │
│                                     │
└─────────────────────────────────────┘
```

**设计要点**：
- 左右对比定性vs量化（用图标区分）
- 四个理由用✅标记，每个理由分两行（理由+推论）
- 最后一条用箭头强调"探索→验证"的逻辑

**演讲提示**：
- （这页是为了回答老师可能的提问，可以快速过）
- "为什么用定性？四个原因：现象新、要过程、要情境、为后续量化打基础。"

## 制作建议

### 软件选择
1. **PowerPoint**（推荐）：
   - 动画效果丰富，易于控制
   - 兼容性好（答辩现场通常是Windows）

2. **Keynote**（如果你用Mac）：
   - 设计更优雅，但要导出为PPT格式

3. **Canva**（在线工具）：
   - 模板丰富，适合快速出成果
   - 但自定义程度较低

### 动画建议
- **第9页（研究模型）**：最重要，必须用动画
  - 先出现自变量
  - 箭头飞入，出现中介变量
  - 箭头飞入，出现因变量
  - 最后从下方飞入调节变量
  - 底部四个命题逐条出现

- **其他页面**：
  - 要点逐条飞入（从左侧或下方）
  - 不要用旋转、跳跃等花哨动画
  - 速度设置为"中速"（不要太慢）

### 配色方案（具体色值）

```css
/* 主色调 */
--primary-blue: #1E3A8A;      /* 深蓝 - 标题 */
--accent-purple: #6366F1;     /* 渐变紫 - 强调元素 */
--accent-cyan: #3B82F6;       /* 蓝色 - 图表 */
--highlight-orange: #F59E0B;  /* 橙色 - 数据/结论 */

/* 背景 */
--bg-light: #F9FAFB;          /* 浅灰白 - 主背景 */
--bg-white: #FFFFFF;          /* 纯白 - 内容框 */

/* 文字 */
--text-dark: #111827;         /* 深灰黑 - 正文 */
--text-medium: #6B7280;       /* 中灰 - 辅助文字 */
```

### 字体大小建议
- **标题**：44pt（加粗）
- **副标题**：32pt（加粗）
- **正文**：20-24pt（常规）
- **注释/引用**：16pt（常规）

### 图标资源
- **Flaticon**（flaticon.com）：免费icon库
- **Noun Project**（thenounproject.com）：简洁图标
- **Emoji**：直接用系统emoji（🤖💡📊等）

## 演讲配合技巧

### 1. 身体语言
- **手势**：讲到关键变量时，指向对应的框图
- **眼神**：看着听众，不要只盯着PPT
- **站位**：站在投影屏幕侧面，不要挡住内容

### 2. 语速和停顿
- **正常语速**：每分钟150-180字
- **关键处放慢**：
  - 核心问题（第5页）
  - 研究模型（第9页）
  - 研究命题（P1-P4）
- **停顿**：
  - 每页切换时停顿1-2秒
  - Part分隔页停顿2秒

### 3. 时间控制
- **总时长**：5-7分钟
- **分配**：
  - Part 1（为什么）：2分钟
  - Part 2（怎么做）：3分钟
  - Part 3（贡献）：1分钟
  - 开场+结尾：1分钟

### 4. 备用策略
- 如果超时：跳过第10、11、13页（变量解释和方法论证）
- 如果时间充裕：在第9页（模型）多停留，详细解释
