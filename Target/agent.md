# Target 文件夹

## 设计目的

存放论文最终产物，是所有工作的汇聚点和长期记忆。

## 内容结构

- `Draft.md` - 论文正文
- 其他支持文件（图表、附录等）

## 使用规则

1. **修改方式**：不直接编辑，通过合并 PR 来修改
2. **一致性**：所有修改须保持论文整体风格和论述逻辑一致
3. **引用规范**：所有引用须符合 APA 第7版规范，来源于 Reference
4. **版本意识**：Target 是持续迭代的产物，每次 PR 合并都是一次演进

## 示例

合并 PR：
1. 打开 PR 文件（如 `PR/PR-0001.md`）
2. 按照 PR 描述的修改位置和内容更新 `Draft.md`
3. 更新 PR 状态为 `merged`
