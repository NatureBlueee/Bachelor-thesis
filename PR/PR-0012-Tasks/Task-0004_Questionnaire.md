# Task-0004: 问卷设计

> **状态**: ⏳ 待开始
> **目标**: 设计作为访谈拓展的问卷

## 设计原则

1. 问卷是访谈的"广度拓展"，不是筛选工具
2. 量表简化，不要太复杂
3. 包含开放题收集定性数据
4. 适合AI社群/LinkedIn投放（+红包激励）

## 问卷结构（草稿）

### Part A: 人口统计 (5题)
- 年龄、行业、工龄、上级年龄段、上级AI熟练度

### Part B: AI素养自评 (6-8题)
- 基于钟柏昌简化版
- 知识/情感/思维各2-3题

### Part C: 感知资源优势 (4-5题)
- 基于RDT核心概念简化
- "我觉得我比上级更懂AI"类

### Part D: 上级开放性 (4题)
- 基于Detert & Burris 4维度
- 每维度1题

### Part E: 向上影响经历 (开放题)
- "请描述一次你用AI知识影响上级决定的经历"
- 是否愿意参与访谈

## 完成标准

- [ ] 量表题目设计
- [ ] 开放题设计
- [ ] 问卷整体结构确认
